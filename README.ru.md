# ✋ GestureLab

**GestureLab** は、ユーザーが手のジェスチャーを使ってパーティクルを**制御・拡大縮小・回転・操作できる**、リアルタイムのタッチレス3Dパーティクルインタラクションプラットフォームです。

![GestureLabDemo](https://github.com/KrishBharadwaj5678/GestureLab/raw/main/GestureLabDemo.png)

## 🧠 機能

| 🚀 機能            | 説明                             |
| ---------------- | ------------------------------ |
| ✨ ハンドジェスチャー検出    | MediaPipeを使用したリアルタイム手の動作トラッキング |
| 🖐️ タッチレス操作      | 画面に触れずにパーティクルを操作可能             |
| 🔄 ダイナミック操作      | パーティクルの回転・拡大縮小・スピン             |
| 🎨 3Dパーティクルエフェクト | 滑らかで没入感のある3Dアニメーション            |
| ⚡ 高性能            | Three.js による最適化されたレンダリング       |
| 🌐 ブラウザベース       | インストール不要でブラウザ上で動作              |

---

| 技術                | 用途              |
