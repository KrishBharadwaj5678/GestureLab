# ✋ GestureLab

**GestureLab** это интерактивная веб-платформа для работы с 3D-частицами в реальном времени без касаний, где пользователи могут **управлять, масштабировать, вращать и манипулировать частицами с помощью жестов рук**.

![GestureLabDemo](https://github.com/KrishBharadwaj5678/GestureLab/raw/main/GestureLabDemo.png)

## 🧠 Возможности

| 🚀 Функция                   | Описание                                                         |
| ---------------------------- | ---------------------------------------------------------------- |
| ✨ Распознавание жестов рук   | Отслеживание движений рук в реальном времени с помощью MediaPipe |
| 🖐️ Управление без касаний   | Управление частицами без прикосновения к экрану                  |
| 🔄 Динамическая манипуляция  | Вращение, масштабирование и управление частицами                 |
| 🎨 3D-эффекты частиц         | Плавные и реалистичные 3D-анимации                               |
| ⚡ Высокая производительность | Оптимизированный рендеринг с использованием Three.js             |
| 🌐 Работа в браузере         | Запускается прямо в браузере без установки                       |
