# ✋ GestureLab

O **GestureLab** é uma plataforma interativa de partículas 3D em tempo real e sem toque, onde os usuários podem **controlar, redimensionar, rotacionar e manipular partículas usando gestos das mãos**.

![GestureLabDemo](https://github.com/KrishBharadwaj5678/GestureLab/raw/main/GestureLabDemo.png)

## 🧠 Funcionalidades

| 🚀 Funcionalidade            | Descrição                                                  |
| ---------------------------- | ---------------------------------------------------------- |
| ✨ Detecção de Gestos Manuais | Rastreamento de gestos em tempo real usando MediaPipe      |
| 🖐️ Controle sem Toque       | Controle de partículas sem tocar na tela                   |
| 🔄 Manipulação Dinâmica      | Rotacionar, redimensionar e girar partículas dinamicamente |
| 🎨 Efeitos de Partículas 3D  | Animações suaves e imersivas de partículas 3D              |
| ⚡ Alto Desempenho            | Renderização otimizada com Three.js                        |
| 🌐 Baseado no Navegador      | Funciona diretamente no navegador, sem instalação          |
